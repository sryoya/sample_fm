# 概要
ItunesAPIベースの音楽検索・試聴サービス


![sicho-kun-demo](https://user-images.githubusercontent.com/14014676/40879075-b5347228-66d5-11e8-95d0-35eceb248055.gif)


# 機能概要
1. アーティスト名・曲名での検索
2. 検索結果の音楽再生(自動で次の曲を再生)
